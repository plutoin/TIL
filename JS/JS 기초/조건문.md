## 조건문
조건의 결과(truthy, falsy)에 따라 다르 코드를 실행하는 구문
- `if`, `else`

```jsx
let isShow = true;
let checked = false;

if (isShow) {
	console.log('Show!'); // Show!
}

if (checked) {
	console.log('Checked!'); 
  // false이므로 콘솔 실행 안 함
}
```

```jsx
let isShow = true;

if (isShow) {
	console.log('Show!'); // Show! 출력
} else {
	console.log('Hide?'); // 거짓이므로 출력 X
}
```
